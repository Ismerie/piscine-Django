Ex01 : Django, framework web.

Dans cette page, nous allons faire une brève présentation de Django et de son historique.

Présentation de Django

Django est un framework web de haut niveau écrit en Python qui encourage le développement rapide et une conception propre et pragmatique. Il est conçu pour permettre aux développeurs de se concentrer sur l'écriture de nouvelles fonctionnalités plutôt que de réinventer la roue. Avec Django, on peut construire des applications web robustes et évolutives plus rapidement et avec moins de code.

Historique de Django

Django a été initialement développé en 2003 par les développeurs Adrian Holovaty et Simon Willison au sein du journal Lawrence Journal-World. Le framework a été rendu open source en juillet 2005 et a depuis été maintenu par la communauté Django et la Django Software Foundation (DSF).

Les points marquants de l'historique de Django incluent :

  • 2003 : Début du développement de Django.
  • 2005 : Django devient open source.
  • 2008 : Création de la Django Software Foundation pour soutenir le développement de Django.
  • 2009 : Sortie de Django 1.0, la première version stable.
  • 2015 : Sortie de Django 1.8, une version LTS (support à long terme).
  • 2017 : Django 2.0 introduit des améliorations significatives et abandonne la compatibilité avec Python 2.
  • 2020 : Django 3.0 continue de moderniser le framework avec des fonctionnalités comme l'async.
  • 2021 : Sortie de Django 4.0 avec de nombreuses nouvelles fonctionnalités et améliorations.

Depuis sa création, Django a gagné en popularité et est utilisé par de nombreuses entreprises et développeurs à travers le monde pour créer des applications web robustes et performantes.
